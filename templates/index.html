Data Requirements

Required File Format:

  • ZIP folder containing file(s) ending in x1dints.fits or .h5

Required Data Columns:

For FITS files:

  • INT_TIMES table with int_mid_MJD_UTC (observation times)
  • EXTRACT1D spectra with: WAVELENGTH, FLUX, and optionally FLUX_ERROR (supported formats: one EXTRACT1D extension per integration, or a single EXTRACT1D table with one row per integration)

For H5 files:

  • Flux: calibrated_optspec, stdspec, or optspec
  • Wavelength: eureka_wave_1d, wave_1d, wavelength, or wave
  • Time: time, bmjd, mjd, or bjd
  • Errors (optional): calibrated_opterr, stdvar, error, or flux_error

Note: Target name, instrument info, and other metadata are optional.
